## Context

Build the features bento grid that showcases BrewClaw's 5 key features with engaging visuals and hover effects. Uses Motion for card hover animations, GSAP for scroll-triggered entrance.

### Design Reference
- bento-grid.tsx from sticky-nav-pricing template
- easyclaw.app bento grid reference
- Dark background section
- Mixed card sizes (2-3-2 pattern or similar asymmetric layout)

## must_haves

- [ ] 5 feature cards in bento grid layout (asymmetric sizing)
- [ ] Persistent Memory card with brain/memory icon
- [ ] Privacy First card with terminal/code visual or animation
- [ ] Always Awake card with clock/24-7 icon
- [ ] Secure by Default card with lock/shield icon
- [ ] Any Messenger card with platform logos (WhatsApp, Telegram, etc.)
- [ ] Motion hover effects on cards (scale, border glow)
- [ ] GSAP ScrollTrigger staggered fade-in for cards
