## Goal

Create a status chip component that displays rotating messages with a pulse-glow animation. The chip sits above the hero section and clicking it scrolls to the batch counter section.

## Must Haves

- [ ] Status chip is visible below navbar, above hero headline
- [ ] 3 messages rotate with smooth transitions
- [ ] Green/amber status dot has pulse-glow animation
- [ ] Clicking chip scrolls to #batch-counter section
- [ ] Animations respect reduced-motion preference

## Tasks

<task id="1">
Create the StatusChip component at `src/components/status-chip.tsx`:
- "use client" directive for client-side state
- Import AnimatePresence and motion from motion/react
- Define messages array with 3 status messages:
  1. "Now accepting Batch 2 applications"
  2. "1,247 assistants deployed this month"
  3. "New: Skills Store now live"
- useState for currentIndex, useEffect with setInterval (4 seconds)
- Render: border chip with flex items-center gap-2
- Status dot (w-2 h-2 rounded-full bg-emerald-500 pulse-glow class)
