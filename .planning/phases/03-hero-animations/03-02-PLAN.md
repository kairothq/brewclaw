## Goal

Create the hero section with two brand animation options: IronClaw-style ASCII shimmer (default) and SplitFlap mechanical flip. Users can switch via URL parameter `?hero=splitflap` for A/B testing.

## Must Haves

- [ ] ASCII "BREWCLAW" animation renders with shimmer effect (default)
- [ ] SplitFlap animation available via ?hero=splitflap URL param
- [ ] Dark background (#0A0A0A) hero section
- [ ] Tagline, sub-copy, CTA button, and trust line all visible
- [ ] CTA links to signup funnel
- [ ] Screen readers get accessible text
