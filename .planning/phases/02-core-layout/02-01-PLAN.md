## Objective

Create a sticky navigation bar with glass blur effect, smooth scroll navigation links, and a mobile-responsive hamburger menu using shadcn/ui Sheet component.

## Context

- Template reference: `/v0 templates/New/sticky-nav-pricing/components/navbar.tsx`
- Use Motion for hover animations (per Phase 1 decision)
- Coffee theme accent color: #78350F (espresso)
- Navigation links: Features, Pricing (Docs disabled with "Coming soon" tooltip)
- CTA: "Get Started" with shimmer effect

<task id="2" type="create">
<description>Add shimmer button CSS animation if not present</description>
<files>
- src/app/globals.css
</files>
<instructions>
Add shimmer-btn class CSS to globals.css:

```css
/* Shimmer animation for CTA buttons */
.shimmer-btn {
  position: relative;
  overflow: hidden;
}

.shimmer-btn::before {
  content: "";
  position: absolute;
  inset: 0;
  background: linear-gradient(90deg, transparent 0%, rgba(255, 255, 255, 0.15) 50%, transparent 100%);
  background-size: 200% 100%;
  animation: shimmer 3s ease-in-out infinite;
}
```

Ensure @keyframes shimmer already exists (it should from Phase 1).
</instructions>
</task>

<task id="3" type="create">
<description>Create Navbar component with glass blur effect</description>
<files>
- src/components/navbar.tsx (new)
</files>
<instructions>
Create navbar based on template with these BrewClaw customizations:

1. **Structure:**
   - Fixed header positioned at top-4 with max-w-3xl centered
   - Glass effect: bg-zinc-900/40 backdrop-blur-md border border-zinc-800
   - Rounded-full shape

2. **Logo (left):**
   - BrewClaw text with font-heading (Space Grotesk)
   - Links to "/" (homepage)

3. **Nav Links (center, desktop only):**
   - Features → scrolls to #features
   - Pricing → scrolls to #pricing
   - Docs → disabled with "Coming soon" tooltip (use shadcn Tooltip)
   - Hover effect: pill background slides using Motion layoutId

4. **CTA (right, desktop only):**
   - "Get Started" button with shimmer-btn class
   - White bg, dark text, rounded-full

5. **Mobile (hamburger icon, md:hidden):**
   - Menu/X toggle icon
   - Opens Sheet component from right side
   - Shows nav links + Get Started button stacked

6. **Animation:**
   - Initial entrance: y: -100, opacity: 0 → y: 0, opacity: 1
   - Hover pill: Motion layoutId for smooth transition

Use "use client" directive for interactivity.
</instructions>
</task>
