## Objective

Create a professional footer with 4-column layout (Product, Company, Legal, Contact) that stacks responsively on mobile, with working email and social links.

## Context

- Reference: easyclaw.app footer style
- Dark background to match site theme
- Coffee accent for hover states (#78350F espresso)
- Copyright with BrewClaw branding
- Social icons: GitHub, X (Twitter), LinkedIn, Instagram

## Tasks

<task id="1" type="create">
<description>Create Footer component with 4-column grid</description>
<files>
- src/components/footer.tsx (new)
</files>
<instructions>
Create a professional footer component:

1. **Container:**
   - Dark background (bg-zinc-950 or bg-background)
   - Full width with max-w-6xl centered
   - Padding: py-16 px-6

2. **Grid Layout:**
   - Desktop: 4 columns (grid-cols-2 md:grid-cols-4)
   - Mobile: 2 columns stacked
   - Gap: gap-8 md:gap-12

3. **Column 1 - Product:**
   - Header: "PRODUCT" (uppercase, tracking-wider, text-sm, font-medium)
   - Links: Features, Pricing, Documentation (disabled), Changelog
   - Links style: text-zinc-400 hover:text-white transition-colors

4. **Column 2 - Company:**
   - Header: "COMPANY"
   - Links: About, Blog (coming soon), Careers (coming soon)

5. **Column 3 - Legal:**
   - Header: "LEGAL"
   - Links: Privacy Policy, Terms of Service, Cookie Policy

6. **Column 4 - Contact:**
   - Header: "CONTACT"
   - Email: hello@brewclaw.com (mailto: link)
   - Location: Austin, TX (static text)

7. **Social Row:**
   - Below columns, centered or left-aligned
   - Icons: GitHub, X, LinkedIn, Instagram
   - Use lucide-react icons (Github, Twitter, Linkedin, Instagram)
   - Icon style: text-zinc-500 hover:text-white transition-colors
   - Size: w-5 h-5

8. **Copyright Row:**
   - Bottom border-t border-zinc-800 separator
   - Text: "© 2024 BrewClaw. Deploy smarter, not harder."
   - Style: text-sm text-zinc-500 text-center py-6

Use "use client" only if needed (likely not needed for static footer).
</instructions>
</task>
