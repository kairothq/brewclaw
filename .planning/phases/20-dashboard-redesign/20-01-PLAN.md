<task type="auto">
  <name>Task 1: Create Zustand dashboard store with thread and credits state</name>
  <files>lib/dashboard-store.ts</files>
  <action>
Create a Zustand store following the exact same pattern as `lib/onboarding-store.ts` (persist middleware, safeSessionStorage fallback for Brave shields). The store manages:

**State:**
- `threads`: Array of `{ id: string, title: string, pinned: boolean, createdAt: Date, updatedAt: Date, messages: Message[] }` where `Message = { id: string, role: 'user' | 'assistant', content: string, createdAt: Date }`
- `activeThreadId`: string | null (currently selected thread)
- `searchQuery`: string (Cmd+K search filter)
- `credits`: `{ balance: number, dailySpend: number, monthlySpend: number }` (default balance: 1.99, spends: 0)
- `sidebarCollapsed`: boolean (default false)
- `creditsOpen`: boolean (credits slide-out panel visibility)
- `settingsOpen`: boolean (settings modal visibility)
- `settingsTab`: 'models' | 'providers' | 'channels' | 'skills' (default 'providers')

**Actions:**
- `createThread()`: Creates new thread with UUID, sets as active, title "New conversation"
- `setActiveThread(id)`: Sets active thread
- `togglePinThread(id)`: Toggles pinned status
- `deleteThread(id)`: Removes thread, clears activeThreadId if it was active
- `addMessage(threadId, message)`: Appends message to thread, updates thread.updatedAt
- `setSearchQuery(query)`: Updates search filter
- `setCredits(credits)`: Updates credit balance/spend
- `toggleSidebar()`: Toggles sidebarCollapsed
- `toggleCredits()`: Toggles creditsOpen
- `openSettings(tab?)`: Opens settings modal, optionally to specific tab
- `closeSettings()`: Closes settings modal

**Selectors (exported functions):**
- `selectFilteredThreads(state)`: Returns threads filtered by searchQuery, sorted into groups: pinned first, then by date (Today, This Month, Older)
- `selectActiveThread(state)`: Returns the currently active thread object
- `selectIsLowBalance(state)`: Returns true if balance < 2.00

Use `crypto.randomUUID()` for IDs. Persist to localStorage with key `brewclaw-dashboard`. Do NOT persist `creditsOpen` or `settingsOpen` (use `partialize` to exclude transient UI state).
  </action>
  <verify>TypeScript compiles without errors: `npx tsc --noEmit lib/dashboard-store.ts` (or check no red squiggles). Store exports useDashboardStore and all selectors.</verify>
  <done>Zustand store exists at lib/dashboard-store.ts with all state, actions, and selectors defined. Transient UI state excluded from persistence.</done>
