## Objective

<instructions>
Add CSS for marquee animation pause behavior:

```css
/* Marquee animation utilities */
@keyframes marquee-left {
  from { transform: translateX(0); }
  to { transform: translateX(-50%); }
}

@keyframes marquee-right {
  from { transform: translateX(-50%); }
  to { transform: translateX(0); }
}

.animate-marquee-left {
  animation: marquee-left var(--marquee-duration, 30s) linear infinite;
}

.animate-marquee-right {
  animation: marquee-right var(--marquee-duration, 30s) linear infinite;
}

/* Pause on hover - container with group class */
.marquee-container:hover .animate-marquee-left,
.marquee-container:hover .animate-marquee-right {
  animation-play-state: paused;
}
```

Place after the existing animation keyframes section.
