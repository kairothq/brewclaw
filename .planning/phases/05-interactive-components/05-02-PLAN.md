## Objective
Create a Skills Store section that displays available AI skills in a filterable grid/carousel format. Inspired by ironclaw.sh skills store visual style. Dark background with category tabs and skill cards.

## Context
- **Visual Reference**: ironclaw.sh skills store (dark theme, grid of skill cards with icons)
- **Animation Provider**: Motion for hover effects, GSAP for scroll entrance
- **Design Tokens**: Dark background (bg-zinc-950), card bg (bg-zinc-900), espresso accents

## Must-Haves (Success Criteria)
- [ ] Dark background section
- [ ] Skills directory header with count badge
- [ ] Category filter tabs (All, Sales, CRM, Browser, etc.)
- [ ] Skill cards with name and description
- [ ] Search bar (visual, links to external skills.sh or placeholder)
