# BrewClaw Font Pairing Options

## Reference: IronClaw.sh uses
- **Headings:** Instrument Serif
- **Body:** Inter
- **Brand:** ASCII block art (░ ▓)

---

## Option A: Modern Tech + Elegant Contrast
**Recommended for BrewClaw**

| Role | Font | Weight | Sample |
|------|------|--------|--------|
| **Display/Brand** | Coolvetica | Regular | `BREWCLAW` |
| **Headlines** | Playfair Display | Italic | *Deploy your assistant* |
| **Body** | Inter | Regular (400) | Clean, readable paragraphs |
| **Code/Technical** | JetBrains Mono | Regular | `npm install brewclaw` |

**Vibe:** Retro-futuristic meets coffee shop elegance. Coolvetica gives a bold, memorable brand presence while Playfair Italic adds sophistication.

## Font Size Scale (Recommended)

```
--text-xs:    12px / 0.75rem   (captions, badges)
--text-sm:    14px / 0.875rem  (secondary text, nav)
--text-base:  16px / 1rem      (body)
--text-lg:    18px / 1.125rem  (lead paragraphs)
--text-xl:    20px / 1.25rem   (card titles)
--text-2xl:   24px / 1.5rem    (section subtitles)
--text-3xl:   30px / 1.875rem  (section titles)
--text-4xl:   36px / 2.25rem   (page titles)
--text-5xl:   48px / 3rem      (hero headlines)

--text-7xl:   72px / 4.5rem    (brand display large)
```

## My Recommendation for BrewClaw

**Option F (Hybrid)** with these specific choices:

1. **Brand Name "BREWCLAW"**: Rendered as ASCII pixel art (like IronClaw) using block characters, with shimmer animation
2. **Section Titles**: Playfair Display Italic - adds elegance, differentiates from competitors
3. **All Other Text**: Geist family - modern, readable, tech-appropriate
4. **Code Snippets**: Geist Mono - consistent with main font family

This gives you:
- ✅ Memorable brand presence (ASCII art)
- ✅ Elegant section breaks (Playfair Italic)
- ✅ Professional body copy (Geist)
- ✅ Technical credibility (Geist Mono)
- ✅ Coffee-shop sophistication vibe
